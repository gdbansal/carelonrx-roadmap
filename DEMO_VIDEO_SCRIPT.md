### Key Points to Highlight

## Recording Tips

### Before Recording
1. **Clear browser cache** and test all features
2. **Prepare demo data**: Create 5-10 sample initiatives with varied priorities
3. **Test login credentials**: Use demo account
4. **Check screen resolution**: Record at 1920x1080
5. **Close unnecessary tabs/applications**

### During Recording
1. **Speak clearly** and at moderate pace
2. **Use mouse highlights** or cursor effects
3. **Pause briefly** between sections
4. **Show loading states** naturally
5. **Zoom in** on important details

### Recommended Tools
- **Screen Recording**: OBS Studio, Camtasia, or Loom
- **Video Editing**: DaVinci Resolve (free) or Adobe Premiere
- **Audio**: Use quality microphone, record in quiet environment
- **Cursor Highlighting**: Use built-in OS features or third-party tools

### Post-Production
1. Add **title cards** for each section
2. Include **text overlays** for key features
3. Add **background music** (subtle, professional)
4. Include **CarelonRx logo** watermark
5. Add **call-to-action** at the end
